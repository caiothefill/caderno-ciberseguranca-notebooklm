# 🔐 Caderno Temático com NotebookLM: Introdução à Cibersegurança

## 🎯 Contexto e Objetivos

Este projeto foi desenvolvido com o objetivo de explorar o uso da Inteligência Artificial como ferramenta de aprendizagem ativa, utilizando o NotebookLM.

O foco foi a introdução à cibersegurança para iniciantes, organizando conteúdos, testando prompts e criando um material de revisão estruturado.

### Objetivos:
- Aprender conceitos básicos de cibersegurança
- Organizar conteúdos com apoio de IA
- Criar um “segundo cérebro” para estudos
- Melhorar o uso de prompts

## 🧠 Engenharia de Prompts e Cicatrizes

### Prompts utilizados:
- Explique o que é cibersegurança de forma simples
- Crie um resumo dos principais conceitos
- Monte um glossário de termos básicos

### Dificuldades encontradas:
- Dificuldade em controlar o comportamento da IA no NotebookLM
- A ferramenta repetia padrões de resposta automaticamente
- Limitação em seguir instruções específicas de personalidade

### Aprendizado:
- Nem toda IA segue exatamente os prompts
- É necessário testar e ajustar constantemente
- O comportamento da IA pode variar conforme a ferramenta

## 📘 Miniguia de Estudo

### 🔹 Resumo

Cibersegurança é a área responsável por proteger sistemas, redes e dados contra ataques digitais.

Principais ameaças:
- Phishing
- Malware
- Ransomware
- Engenharia social

Boas práticas:
- Usar senhas fortes
- Não clicar em links suspeitos
- Ativar autenticação em dois fatores

---

### 🔹 Glossário

- Cibersegurança: proteção de sistemas e dados  
- Phishing: golpe para roubar informações  
- Malware: software malicioso  
- Firewall: controle de acesso de rede  
- Vulnerabilidade: falha explorável  

---

### 🔹 Prompts reutilizáveis

- Explique [conceito] de forma simples  
- Resuma este conteúdo em tópicos  
- Crie um glossário sobre o tema  
- Gere perguntas de revisão
